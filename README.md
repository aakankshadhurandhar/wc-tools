
# wc-tool

Our own version of the Unix command line tool wc.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd wc-tools
```

Install dependencies

```bash
  mvn clean package
  source .bash_profile
```

Start the server

```bash
  ccwc --c file_name #count no of bytes of file
```
